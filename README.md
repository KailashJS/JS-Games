# Developed the simple but Famous games using Javascript

## List of Games
* Memory Game 
* Whack-a-mole
* Connect Four
* Nokia 3310 Snake
* Space invaders
* Frogger
* Tetris

# Core JavaScripts Concepts involved are
## Memory Game
* push()
* querySelector()
* setAttribute()
* getAttribute()
* appendChild()
* Math.random()
* sort()
* For Loops 
* createElement()

## Whack-a-mole 
* querySelector()
* addEventListener()
* setInterval()
* classList()
* forEach()
* Arrow functions

## ConnectFour 
* querySelector()
* addEventListener()
* onClick()
* classList.contains()
* classList.add()
* For Loops 
* Arrow functions
